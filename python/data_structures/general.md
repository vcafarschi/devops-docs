# Strings
- strings can be indexed
  - stringvar[0]
- negative indexing
  - stringvar[-1]

# Slice a sequence
- start, stop, step value
  - stringvar[0:6]
  - stringvar[0:6:2]
  - stringvar[-4:-2]
  - stringvar[::-1] return a reversed sequence
