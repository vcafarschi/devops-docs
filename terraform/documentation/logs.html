Terraform Logs

Terraform has detailed logs which can be enabled by setting the TF_LOG environment variable

  • TF_LOG=TRACE
  • TF_LOG=DEBUG
  • TF_LOG=INFO
  • TF_LOG=WARN
  • TF_LOG=ERROR
  • TF_LOG=JSON - outputs logs at the TRACE level or higher, and uses a parseable JSON encoding as the formatting.

Logging can be enabled separately for:

  • Terraform Core using TF_LOG_CORE=smth environment variable
  • Terraform Providers through TF_LOG_PROVIDER=smth environment variable

As well you can save the log in the separate file specified by TF_LOG_PATH environment variable

WARNING !!! If terraform plan is running slowly, turn off trace-level logging and consider increasing parallelism (-parallelism=n).
